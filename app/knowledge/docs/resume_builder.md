# Resume Builder on ZyncJobs

## What is Resume Builder?
Resume Builder is a guided tool that helps you create a professional, ATS-friendly resume in minutes. It provides templates, suggestions, and real-time ATS scoring as you build.

## How to Create a Resume
1. Go to Resume Builder from your candidate dashboard.
2. Choose a template style (Modern, Classic, Professional, Creative).
3. Enter personal details: name, email, phone, LinkedIn, portfolio.
4. Add work experience with company name, role, dates, and responsibilities.
5. List education: degrees, institutions, graduation years.
6. Add skills with proficiency levels.
7. Include certifications, projects, and languages.
8. Preview and download as PDF.

## Key Features
- Auto-save as you type.
- AI-powered content suggestions for bullet points.
- ATS score displayed in real time.
- Keyword recommendations based on target job.
- Multiple resume versions for different job applications.
- Export to PDF or DOCX format.

## Tips
- Use action verbs like Led, Developed, Implemented.
- Quantify achievements with numbers and percentages.
- Keep each resume focused on a specific role.
- Update your resume regularly with new skills and experience.
