# Skill Assessment on ZyncJobs

## What is Skill Assessment?
Skill Assessment is a feature where candidates take tests to evaluate technical skills, soft skills, aptitude, and domain-specific knowledge. Results help employers shortlist candidates faster.

## Available Assessments
- Technical Skills: Python, Java, JavaScript, SQL, Data Structures, Algorithms
- Soft Skills: Communication, Problem Solving, Teamwork
- Aptitude: Numerical Reasoning, Verbal Ability, Logical Reasoning
- Domain Specific: Marketing, Finance, Healthcare, IT Support

## How It Works
1. Go to the Skill Assessment page in your candidate dashboard.
2. Choose an assessment from the available list.
3. Read instructions and time limit before starting.
4. Answer multiple-choice, coding, or descriptive questions.
5. Submit the test on completion.
6. View your score and percentile ranking immediately.

## Scoring and Results
- Each test is scored out of 100.
- Results are shared with employers you apply to.
- You can retake assessments after 30 days.
- High scores earn a Verified Skill Badge on your profile.

## Tips for Candidates
- Complete assessments in a quiet environment.
- Read questions carefully before answering.
- Manage time efficiently; move on if stuck.
- Practice with sample questions available on ZyncJobs.
